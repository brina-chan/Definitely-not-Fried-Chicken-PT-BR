## 🛠️ Instalação

Siga os passos abaixo e **certifique-se de que o jogo esteja fechado** antes de começar.

1. **Localize a pasta do jogo:** Vá até a pasta onde o jogo está instalado (geralmente dentro da pasta da Steam ou Epic Games).
2. **Acesse os dados:** Abra a pasta `DNFC_Data`.
3. **Backup de Segurança:** Localize o arquivo `resources.assets` e faça um backup dele (ex: renomeie para `resources.assets.original`).
4. **Substitua o arquivo:** Copie o arquivo `resources.assets` que você baixou aqui e cole na pasta `DNFC_Data`.
5. **Confirme a substituição:** Quando o Windows perguntar, escolha "Substituir o arquivo no destino".
6. **Inicie o jogo:** No menu principal, selecione o idioma **Inglês** (a tradução substitui esse slot).

## ⚠️ Observações

* **Build Compatível:** Tradução testada na versão **1.0.45**. Em versões muito antigas ou futuras atualizações grandes, podem ocorrer erros.
* **Backup:** Nunca esqueça de fazer o backup do arquivo original para evitar ter que reinstalar o jogo caso algo dê errado.
* **Slot de Idioma:** Lembre-se que você deve selecionar **Inglês** nas opções do jogo para ativar o PT-BR.
